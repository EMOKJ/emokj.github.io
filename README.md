## 部署

本项目使用GitHub Pages进行部署。

### 本地运行

1. 克隆仓库
```bash
git clone https://github.com/your-username/emo-vision-site.git
```

2. 打开 `index.html` 文件

### GitHub Pages部署

1. 将代码推送到GitHub仓库
2. 进入仓库设置
3. 找到 "Pages" 选项
4. 选择 "main" 分支
5. 保存设置

网站将在 `https://your-username.github.io/emo-vision-site/` 上线。

## 自定义

### 修改下载链接

在 `index.html` 中找到下载按钮，将 `href="#"` 替换为实际的下载链接。

### 添加截图

将截图放入 `images/` 目录，然后在HTML中引用。

### 修改联系方式

在联系区域修改GitHub、Telegram等链接。
